How to run the program:
1. Run the server 'ScheduleMaker.js' on the terminal
2. On the browser, run the application using 'localhost:8080/index.html'
3. Add courses and select filters to generate schedules


The code written by the team:
- Client Side:
    - index.html
    - css/main.css
    - js/inputs.js
- Server Side:
    - ScheduleMaker.js
    - TMSParser.js
- The rest are from Bootsrap templates and NodeJs modules
